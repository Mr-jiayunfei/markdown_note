# C#调用C++动态库

If the C++ library function returns `char*`, C# code will treat it as `IntPtr` and `Marshal.PtrToStringAnsi()` will convert it into C# `string`.

```
IntPtr verText = f1();
string s = Marshal.PtrToStringAnsi(verText);
Console.WriteLine(s);
```
